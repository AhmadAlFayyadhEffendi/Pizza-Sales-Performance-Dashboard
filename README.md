# Pizza-Sales-Performance-Dashboard
## Project Overview
This project features a comprehensive Pizza Sales Performance Dashboard built in Microsoft Excel. It is designed to track key business metrics and analyze sales trends to help a pizza restaurant optimize its operations and marketing strategies.
## Key Performance Indicators (KPIs)
The dashboard monitors three primary metrics:
- Total Revenue: $817,860.05.
- Total Orders: 21,350.
- Average Order Value (AOV): $38.31.
## Key Insights & Visualizations
- Top 10 Best Sellers: A bar chart identifying the most popular pizzas, led by 'The Classic Deluxe Pizza' with 2,453 units sold.
- Revenue by Category: A stacked bar chart breaking down sales by pizza category (Veggie, Supreme, Classic, Chicken) and size (L, M, S, XL, XXL).
- Quantity per Category: A pie chart showing the percentage distribution of sales, where the 'Classic' category holds the largest share at 30%.
- Monthly Revenue Trend: A line chart tracking financial performance from January to December, showing peak revenue in July ($72,557.90).
## Interactive Features
- Dynamic Filtering: The dashboard includes multiple Slicers to filter data by order_date (Months), pizza_size, pizza_category, and specific pizza_name.
## Tools Used
- Microsoft Excel: Advanced Pivot Tables, Pivot Charts, Slicers, and Data Visualization.
